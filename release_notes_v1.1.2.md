⚠️ **서명되지 않은 앱 실행 안내 (보안 경고 우회):**
앱 설치 후 실행 시 "악성 코드가 없음을 확인할 수 없습니다" 경고가 뜨면, 터미널(Terminal)을 열고 아래 명령어를 입력해 다운로드 격리(Quarantine) 속성을 강제로 해제해 주세요.
```bash
xattr -cr /Applications/NotchPlay.app
```
(또는 마우스 우클릭 후 '열기'를 선택하셔도 됩니다.)
